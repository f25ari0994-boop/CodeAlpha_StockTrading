# CodeAlpha_StockTrading
This Java console project simulates a basic stock trading system. Users can view market stocks, buy and sell shares, and track their portfolio. It uses OOP concepts like classes and objects with ArrayList for data management, demonstrating real-world trading logic in a simple way.

#Stock Trading Platform
#Project Description

The Stock Trading Platform is a Java console-based application that simulates a simple stock market system. It allows users to view available stocks, buy and sell shares, and manage their portfolio. The project demonstrates real-world trading logic using Object-Oriented Programming (OOP) concepts.

#Features
View market stocks with prices
Buy stocks and add to portfolio
Sell stocks from portfolio
Track portfolio holdings
View transaction history (buy/sell)
#Technologies Used
Java
Object-Oriented Programming (OOP)
ArrayList
Console-based input/output
#How It Works
The system displays available stocks in the market
User selects a stock to buy or sell
Portfolio is updated dynamically
Transactions are recorded in history
User can view portfolio at any time
#Project Structure
Stock Class → Stores stock details (name, price)
User Class → Manages portfolio and actions
Transaction Class → Records buy/sell history
#How to Run
Compile the program:
javac StockTradingPlatform.java
Run the program:
java StockTradingPlatform
#Learning Outcome

This project helps in understanding:

OOP concepts (classes, objects)
Data handling using ArrayList
Real-world system simulation
Console-based application design
